Tagix Browser

Tagix Browser is a web browser created specially for KolibriOS.

It renders local and remote pages through the Tagix HTML/CSS pipeline and hosts page content inside the native KolibriOS UI.

The browser shell is loaded from assets/shell.html, and the current page is displayed in a dedicated document view below the toolbar.

This built-in page is routed through tagix://welcome and loaded from assets/pages/welcome.html.

What To Test

  • Semantic shell tags: header, nav, section, h1, small, iframe
  • Shell styling through inline HTML5 style attributes
  • Inline links, lists, headings and forms in rendered pages
  • Document focus, hover and scroll inside the hosted page view
Architecture

shell.html -> Parse -> ui.DocumentNode -> DocumentView

Tagix Browser is intended to be a native KolibriOS browser, not just a demo page viewer.
