## Ebitengine Ports

This directory contains KolibriOS ports of games that were originally built with Ebitengine/Ebiten.

Current ports:

- `zombic` - top-down zombie shooter, ported from `musztardem/zombic`
- `zarulem` - simple racing game, ported from `huaesthetica63/ZaRulem`

Each app keeps its own `Makefile`, `main.go`, and runtime `assets/` directory.
